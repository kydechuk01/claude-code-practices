# MCP-серверы — подключение к внешним системам

> Этот раздел для продвинутых пользователей. Если вы не знаете, что такое PostgreSQL или Playwright — можете пропустить его и вернуться позже.

## Суть
MCP (Model Context Protocol) — расширения, которые дают Claude Code прямой доступ к внешним системам: браузерам, базам данных, документации. Без них AI работает только с файлами на диске. С ними — может открыть сайт, выполнить SQL-запрос, проверить актуальную документацию.

## Ключевые серверы

### Context7 — актуальная документация
**Зачем:** Claude Code обучен на данных до определённой даты. Библиотеки и фреймворки обновляются, API меняются. Без Context7 AI может предложить устаревший метод или несуществующую функцию. С Context7 — подтягивает документацию конкретной версии библиотеки в реальном времени.

**Как работает:** вы спрашиваете «как сделать X в библиотеке Y» — Claude Code обращается к Context7, получает актуальную документацию и отвечает на основе неё, а не на основе своей памяти.

**Рекомендация:** установить глобально для всех проектов. Это самый полезный MCP-сервер для предотвращения галлюцинаций о библиотеках.

### Browser (Playwright) — управление браузером
**Зачем:** AI может открыть веб-страницу, сделать скриншот, нажать кнопку, заполнить форму — всё автоматически. Это открывает мощный паттерн: Visual QA Loop.

**Visual QA Loop:**
```
Изменение → Сборка → Скриншот в браузере → Визуальная проверка AI → Итерация
```

AI сам проверяет результат визуально: правильно ли выглядит страница, не сломалась ли вёрстка, корректны ли данные. Без участия человека.

**Применение за пределами разработки:**
- Скриншоты внешних сайтов для анализа (конкуренты, источники данных)
- Автоматическое заполнение форм
- Извлечение данных со страниц

### PostgreSQL / базы данных
Прямые SQL-запросы к базе. AI может сам проверить данные, не дожидаясь, пока вы скопируете результат запроса.

### GitHub
PR, issues, push — интеграция с рабочим процессом разработки.

### Filesystem
Прямой доступ к файлам за пределами рабочей директории.

## Как установить MCP-сервер

Попросите Claude Code:

> Установи MCP-сервер Context7 для работы с документацией библиотек.

Или:

> Установи MCP-сервер Browser (Playwright) для управления браузером и скриншотов.

Claude Code выполнит установку и настройку.
