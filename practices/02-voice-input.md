# Голосовой ввод — Handy + Whisper Large

## Инструмент
- **Handy** — десктопное приложение для голосового ввода
- https://handy.computer/ (скачивание: нужен VPN из РФ)
- **Whisper Large** (1 ГБ) — модель распознавания речи

## Почему Whisper Large, а не GigaAM
GigaAM v3 точнее на чистом русском, но ломается на английских терминах в русской речи. Whisper Large распознаёт смешанный русский+английский без ошибок. Разница в скорости вывода 1-2 секунды — некритично.

## Настройка
- Настраивается за 1 минуту
- Сочетание клавиш: выбрать не конфликтующее с вашим редактором (например, CTRL+1)
- HotkeyDetective.exe — утилита для проверки конфликтов сочетаний
- Режим: включение → надиктовка → выключение (НЕ push-to-talk)
- Периодически Handy теряет сочетание → перезапустить

## Практики использования
- Короткие промпты — диктовать прямо в консоль Claude Code
- Длинные промпты (несколько абзацев) — диктовать в текстовый редактор, потом вставлять или сохранить в файл и дать AI ссылку на файл
- Консоль может обрезать длинный текст, и AI начнёт выполнять неполный промпт
- В Handy есть история надиктовок — можно скопировать последнюю
- Проговаривайте чётко окончания слов
- Whisper фильтрует фоновые разговоры, но при паузах может захватить чужую речь

## Артефакт Whisper
Whisper вставляет в конце фразы "Продолжение следует...", "Редактор субтитров..." — вы быстро привыкнете удалять это на автомате.

## Выгода
- Скорость ввода x3-x5
- Вы перестаёте упрощать задачу "потому что лень печатать"
- AI получает больше контекста → результат лучше
- Привыкаете за 2 дня, начинаете использовать везде (чаты, поисковики)
