# Кастомные агенты

Агент — md-файл в `~/.claude/agents/` с описанием роли. AI работает в рамках заданной роли, а не как генералист.

**Ключевые принципы:**
- Агентов не нужно планировать заранее — создаются на лету из накопленного контекста
- В реальных проектах обычно 5-15 агентов (универсальные + проектные)
- Агенты — не слепые исполнители: если задача некорректна или есть лучший вариант — агент скажет. Это заложено в конструкцию, а не является побочным эффектом
- Тиринг моделей: opus для анализа, sonnet для рутины, haiku для проверок

Как создать агентов под ваши задачи: `recipes/create-agents.md`
Агенты-персоны для UX-ревью и проверки бизнес-гипотез: `recipes/user-personas-as-agents.md`
