# Мультиагентная работа

Несколько агентов с разной экспертизой работают над одной задачей. Три паттерна: параллельный анализ + синтез, перекрёстная проверка, волновая обработка.

Как организовать коллегиальную работу: `recipes/multi-agent-collaboration.md`

## Дополнительные практики

### TDD-воронка (для разработчиков)
3 агента параллельно (QA, разработчик, дизайнер) генерируют ~200 тестов → синтез и дедупликация → ~100 тестов → организация в файлы. Всё ДО написания кода. Тесты определяют контракт.

Подробнее: `recipes/tdd-for-developers.md`

### Тихий режим (Silent Mode)
AI работает автономно без участия оператора: получает все разрешения заранее, ведёт протокол, выдаёт отчёт по завершении. В реальных проектах десятки фоновых агентов работают без вмешательства.

Подробнее: `recipes/create-skills.md` (секция Silent Mode)

### Мега-сессии
Сессии могут длиться неделями: сотни субагентов, десятки компрессий контекста. Сессия возобновляется (resume), контекст сохраняется. При исчерпании окна — автоматическая компрессия.

### Прототипирование
Множество HTML-прототипов из одного промпта — для согласования со стейкхолдерами. Визуальные макеты, которые можно открыть в браузере и обсудить.
