# Система памяти — три слоя

Claude Code забывает всё после сессии. Система памяти решает это тремя слоями:

1. **MEMORY.md** — загружается автоматически (~200 строк). Схема данных, ключевые решения, текущий статус.
2. **Reference files** — подробные документы по темам. AI подтягивает по необходимости, не забивая контекст.
3. **Карточки ловушек** — подробные записи решённых проблем в `_docs/gotchas/` (Проблема → Причина → Решение → Предотвращение). Краткий реестр — в таблице «Известные ловушки» в CLAUDE.md (до 20 строк, загружается автоматически). Ошибки не повторяются.

Карточки ловушек — как командная wiki постмортемов, которая автоматически загружается в каждую сессию.

Как настроить: `recipes/setup-memory.md`
